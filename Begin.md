## Installer la derniere version nvm 

```bash
nvm install 22
```

## Installer angular 

```bash
npm install -g @angular/cli
```

## Crer un projet angular

```bash
ng new nomduprojet
```

## Installer angular material

```bash
ng add @angular/material
```

## Lancer le server

```bash
ng serve
```
